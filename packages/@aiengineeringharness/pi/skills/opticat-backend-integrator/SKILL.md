# OptiCat Backend Integrator

## Overview

This skill provides specialized knowledge and tools for interacting with the backend systems of the OptiCat platform. It focuses on integrating with existing services, manipulating data at a deeper level, and understanding system configurations, particularly for components that might be implemented in lower-level languages like C/C++.

## Core Capabilities

### 1. Backend API Interaction
Guidance on making direct calls to OptiCat's backend APIs, potentially including undocumented or lower-level interfaces.
- **Reference**: `references/backend-api-specs.md` for detailed API specifications and protocols.

### 2. C/C++ Integration & FFI
Instructions and examples for interfacing with C/C++ components of the OptiCat backend using Foreign Function Interface (FFI) or inter-process communication (IPC) mechanisms.
- **Reference**: `references/ffi-guidelines.md` for best practices and common patterns for C/C++ interaction.

### 3. Data Manipulation & Database Access
Provides patterns for direct database interaction or using backend services to manipulate OptiCat data, ensuring data integrity and consistency.
- **Reference**: `references/opticat-db-schema.md` for database schema and data models.

### 4. System Configuration Management
Guidance on reading, modifying, and validating OptiCat backend configurations.

### 5. Error Handling & Diagnostics
Best practices for debugging backend issues, interpreting logs, and implementing robust error handling strategies.
