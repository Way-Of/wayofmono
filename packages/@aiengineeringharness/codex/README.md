# Codex — AI Engineering Harness Integration

[Codex CLI](https://github.com/openai/codex) is OpenAI's agentic coding terminal tool.

## Structure

```
codex/
├── agents/           Agent definitions (snake_case .md files)
├── rules/            Codex rules for instruction overrides
└── skills/           Generated skill files (skill.yaml + prompt.md per skill)
```

## Installation

Use the shared installer or setup script:

```bash
ai-harness --tool=codex          # Download install
./setup.sh codex                 # Symlink via GNU Stow
```

## Codex-Specific Notes

- **Project memory**: Codex uses `RULES.md` in the project root (generated by `/init`)
- **Skills**: Loaded from `skills/<name>/skill.yaml` + `prompt.md`
- **Agents**: Snake_case naming convention in `agents/`
- **Config location**: `~/.antigravity/` → symlinked to this directory
- Commands: Codex does not have a native slash command system; workflows are driven via `rules/` overrides and skill prompts
