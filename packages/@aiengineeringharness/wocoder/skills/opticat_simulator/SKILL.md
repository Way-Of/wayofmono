# OptiCat Simulator

## Overview

This skill provides specialized knowledge and tools for setting up, executing, and analyzing simulations related to OptiCat's HVAC domain. It aims to streamline the process of configuring simulation parameters, running various simulation scenarios, interpreting the results, and performing what-if analysis to evaluate system performance and efficiency.

## Core Capabilities

### 1. Simulation Setup
Guidance on defining simulation environments, including parameters for HVAC system components, environmental conditions, and operational schedules.
- **Reference**: `references/simulation-config-guide.md` for parameter definitions and configuration examples.

### 2. Simulation Execution
Instructions and examples for executing simulation runs using OptiCat's simulation engine or external tools.
- **Script**: `scripts/run_simulation.sh` (placeholder) for automating simulation execution.

### 3. Results Analysis
Provides methods for interpreting simulation outputs, extracting key performance indicators (KPIs), and generating reports.
- **Reference**: `references/simulation-output-formats.md` for expected data structures and analysis techniques.

### 4. What-If Analysis
Guidance on modifying simulation parameters to explore different scenarios and evaluate their impact on HVAC system performance.

### 5. Integration with OptiCat Backend
Consideration for how simulation results or configurations can be integrated back into OptiCat's backend systems.
