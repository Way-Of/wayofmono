# OTel Transform Language (OTTL)

This skill provides expertise in writing OTTL expressions to modify telemetry data in flight within the Collector.

## Use Cases
- **Redaction:** Stripping sensitive headers or tokens from spans.
- **Attribute Mapping:** Renaming legacy attributes to match new semantic conventions.
- **Filtering:** Dropping noisy or redundant telemetry.
