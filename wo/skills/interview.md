# Interview Skill

This skill allows an agent to "grill" the user or another agent about a design or plan to uncover hidden risks or ambiguities.

## Goal
Reach a shared, deep understanding of the problem and the proposed solution.

## Strategy
- **Drill Down:** Ask "Why?" multiple times to find root motivations.
- **Edge Cases:** Question how the system handles errors, timeouts, or scale.
- **Assumptions:** Identify and challenge unspoken assumptions.
- **Branching:** Walk down every decision tree until a leaf is reached.
