# /wom-lens-widget

Show or hide the persistent Wom-Lens integrity widget at the bottom of the TUI.

## Visuals
The widget displays:
- Diagnostic counts (🔴 Errors, 🟡 Warnings).
- Active runner status.
- Language server initialization progress.
